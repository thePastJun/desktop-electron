# desktop-electron

## 安装依赖
```
npm install
```

###  纯vue项目启动
```
npm run serve
```

###  纯vue项打包
```
npm run build
```
### eslint修复文件
```
npm run lint
```
## 桌面应用 
### 编译并启动APP
```
npm run electron:serve
```
### 打包app
```
npm run electron:build
```
